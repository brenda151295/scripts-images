resize.py

FILE -- Name of file to resize (this file must be at the same level as resize.py)
WIDTH -- Redimension width

crop.py

FILE -- Name of file to resize (this file must be at the same level as crop.py)
HEIGHT -- Cutting height
WIDTH -- Cutting width
FORMAT -- Format of the images

* You will find the transformed images in the crop/ folder
